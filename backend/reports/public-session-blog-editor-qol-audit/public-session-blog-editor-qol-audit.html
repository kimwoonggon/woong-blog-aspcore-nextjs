Public Session + Blog Editor QoL Audit

Date: 2026-04-21

This focused PR cleans public admin affordance session handling, preserves contact authored content, and improves BlogEditor save/excerpt ergonomics.

Changed

  • Removed the redundant public layout session fetch and unused Navbar session prop.
  • Centralized public admin affordance visibility in src/lib/auth/public-admin.ts.
  • Updated public blog, works, contact, introduction, and resume pages to use the shared gate.
  • Removed contact fallback direct-email injection.
  • Added BlogEditor top quick save/back actions, kept bottom Create/Update submit, and added horizontal workspace resizing.
  • Kept excerpt-aware create/update/inline update flows.
  • Changed browser API default to same-origin /api for CSP-compatible Docker mutations.

  • Anonymous public read access.
  • Excerpt-first card presentation.
  • Mobile/feed/search redesign, broad density tuning, related-content refactors, and autosave debounce redesign.
